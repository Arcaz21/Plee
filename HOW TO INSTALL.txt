HOW TO INSTALL
I. Install XAMPP
	1. Download XAMPP
	2. Complete Installation
II. Copy Plee Files
	1. Download files from GITHUB
	2. Create Folder "Plee" on C:\xampp\htdocs
	3. Copy "controllers, models and views" from downloaded files to C:\xampp\htdocs\Plee
III. Copy Configuration Files
	1. copy Windows folder to C:/ (Provide administrator rights if prompted)
	2. copy XAMPP folder to C:/
IV. Copy Database
	1. Open localhost/phpmyadmin
	2. Create new database "plee"
	3. Import plee.sql from folder to plee database on phpmyadmin
	4. Click Go wait for the databse to be successfulss
V. Plee
	1. Open browser and type Plee.com
	2. Try to Register and to Login
